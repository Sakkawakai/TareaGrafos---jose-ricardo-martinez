Codigo desarrollado por Jose Ricardo Martinez 

propósito: código de detección de rutas, para encontrar la mas corta posible.

para clase de Estructura de Datos

instrucciones.
1. descargar Mn.Py (archivo principal)
2. descargar SGdtt. Py (archivo de cargar/guardar)
3. obligatorio descargar mapa.jpg, o una imagen con el nombre "mapa.jpg" para que pueda funcionar el script
4. (opcional) descargar MapSave.txt para cargar las ubicaciones y rutas predefinidas.
